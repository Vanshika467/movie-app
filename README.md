# 🎬 MovieHub

A modern movie browsing web app built using React and TMDB API.

## 🚀 Features

- 🔍 Search movies (debounced search)
- 🎬 Popular movies list
- ❤️ Add to favorites (localStorage)
- 📄 Movie details page (rating, genres, cast)
- ♾️ Infinite scroll
- ⏳ Loading spinner
- ❌ Error handling
- 🎨 Clean Netflix-style UI

## 🛠️ Tech Stack

- React (Vite)
- React Router
- TMDB API
- CSS (Flexbox)

## 📦 Installation

```bash
git clone https://github.com/your-username/movie-app.git
cd movie-app
